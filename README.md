# TaskFlow

TaskFlow es una aplicación web de gestión de tareas creada con **HTML, CSS y JavaScript**. Permite a los usuarios añadir, eliminar y buscar tareas de manera dinámica.

Las tareas se guardan en el **LocalStorage del navegador**, por lo que no se pierden al refrescar la página.

---

## Características

* Añadir tareas con nombre, categoría y prioridad.
* Eliminar tareas fácilmente.
* Buscar tareas mediante un filtro de texto.
* Persistencia de datos usando LocalStorage.
* Diseño responsive que funciona en móviles y ordenador.

---

## Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript
* LocalStorage

## Cómo usar la aplicación

1. Escribe una tarea en el campo de texto.
2. Selecciona la categoría y prioridad.
3. Pulsa **Añadir tarea**.
4. Puedes eliminar tareas usando el botón **Eliminar**.
5. Usa el buscador para filtrar tareas.
